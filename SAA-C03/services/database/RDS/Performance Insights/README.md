## Performance Insights

- RDS / Aurora 의 데이터베이스 성능을 분석하는 모니터링 기능

---

## 핵심 개념

- DB의 "병목 원인"을 시각적으로 분석
- 어떤 쿼리가 느린지 확인 가능

---

## 특징

    - 시각화된 성능 분석
    - 쿼리 단위 분석
    - 병목 구간 식별 가능
    - AWS에서 자동 수집

---

## 주요 기능

### 1️⃣ DB Load 분석

- DB의 부하를 시각적으로 표시

👉 예:
- CPU 사용
- I/O 대기
- Lock 대기

---
