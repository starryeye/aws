## Athena

- AWS에서 제공하는 서버리스 SQL 쿼리 서비스
    - S3에 저장된 데이터를 직접 조회

---

## 핵심 개념

- 데이터베이스 없이 SQL로 분석 가능
- S3 데이터를 그대로 조회

---

## 특징

    - 서버리스 (Serverless)
    - 인프라 관리 불필요
    - SQL 기반 분석
    - 사용한 데이터만큼 과금 (Pay-per-query)

---

## 동작 방식

    S3 (데이터 저장)
        ↓
    Athena (SQL 실행)
        ↓
    결과 반환

---

## 특징 정리

- 데이터 저장 ❌ (S3 사용)
- 분석만 수행
- 빠르게 쿼리 가능

---

## 주요 기능

### 1️⃣ Schema-on-Read

- 데이터를 미리 구조화할 필요 없음
- 읽을 때 스키마 적용

---

### 2️⃣ 다양한 데이터 형식 지원

- CSV
- JSON
- Parquet
- ORC

---

### 3️⃣ Glue Data Catalog 연동

- 테이블 메타데이터 관리

---

## 사용 케이스

- 로그 분석
- Ad-hoc 쿼리
- 빠른 데이터 조회
- BI 분석

---

## Athena vs Redshift vs EMR

| 구분 | Athena | Redshift | EMR |
|---|---|---|---|
구조 | 서버리스 | 클러스터 | 클러스터 |
데이터 | S3 | 내부 저장 | S3/HDFS |
용도 | 즉석 쿼리 | 분석 | 처리 |
관리 | 없음 | 있음 | 있음 |
