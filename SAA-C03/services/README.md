## AWS Services

- EC2
    - Elastic Compute Cloud
    - Virtual servers in the cloud (compute service)

- RDS
    - Relational Database Service
    - Managed relational database service (MySQL, PostgreSQL, etc.)

- S3
    - Simple Storage Service
    - Object storage service for files and backups

- Route 53
    - Scalable DNS service
    - Domain registration and traffic routing

- ELB
    - Elastic Load Balancing
    - Distributes incoming traffic across multiple targets (EC2, containers, etc.)

- CloudFront
    - Content Delivery Network (CDN)
    - Global content caching and delivery

- Lambda
    - Serverless compute service
    - Run code without provisioning or managing servers
