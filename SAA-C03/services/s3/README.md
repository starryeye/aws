## S3

- S3 유형
  - S3 Standard
    - 자주 접근 시, 즉시 조회

  - S3 Standard-IA
    - 거의 접근 없음, 즉시 조회, 다중 AZ

  - S3 One Zone-IA
    - 거의 접근 없음, 즉시 조회, 단일 AZ

  - S3 Glacier Instant Retrieval
    - 거의 접근 없음, 즉시 조회, 법/감사/규정 목적

  - S3 Glacier Flexible Retrieval
    - 거의 접근 없음, 조회 시 1시간 소요, 법/감사/규정 목적

  - S3 Glacier Deep Archive
    - 거의 접근 없음, 조회 시 12시간 소요, 법/감사/규정 목적

  - S3 Intelligent-Tiering
    - 불규칙한 접근

- S3 버전 관리
  - 기본적으로 파일은 덮어쓰기 되어버린다.
  - 버전 관리 기능을 활성화 하면 이전 버전의 파일까지 보존된다.

- S3 객체(파일) 잠금
  - 일정 기간 동안 파일의 수정/삭제를 차단하는 기능
  - 모드
    - 거버넌스 모드
      - 관리자를 제외한 모든 사용자 수정/삭제 차단
    - 규정 준수 모드
      - 모든 사용자 수정/삭제 차단
  - 법적 보존 기능
    - 기간에 상관없이 수동 해제 전까지 수정/삭제 차단

- S3 수명 주기 정책
  - 파일을 N일 이후 자동으로 이동시키거나 삭제하는 기능

- S3 암호화 SSE
  - 서버가 자체적으로 파일을 암호화한다.
  - 종류
    - SSE-S3
      - S3가 key를 생성하고 관리
    - SSE-KMS
      - AWS KMS가 key를 생성하고 관리
      - key를 주기적으로 자동 교체
    - SSE-C
      - 사용자가 직접 key를 생성하고 관리

- S3 Batch Operations
  - S3에 저장된 파일들에 대해 동일한 작업을 한번에 실행시킬때 사용
